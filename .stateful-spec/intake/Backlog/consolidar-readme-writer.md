# Problema

A iteração [006-readme-writer](../../history/006-readme-writer.md) produziu a
pesquisa pura da persona especialista em escrita de README files no vault Jandi
(`Agente - Persona Especialista em Escrita de README Files.md`), validada pelo
usuário, mas o catálogo ainda não tem a skill `readme-writer` — o entregável
final da linha O-003, adiado explicitamente para iteração futura no encerramento
da 006.

## Ação

Consolidar a pesquisa como skill no catálogo, nos moldes das demais:

- Derivar do doc do vault um `readme-writer/SKILL.md` (EN) — os três ofícios
  (captar fundamentos, projetar a comunicação, manter o registro vivo) como
  corpo procedimental, com gatilho (`description`) passando G1–G3.
- Redigir o `readme-writer/README.md` (docs voltados a humanos) e atualizar a
  tabela do README raiz.

## O que não faremos agora

- Não alterar as skills existentes.
- Não criar a contraparte PT-BR (registrada como possibilidade na 006, sem
  compromisso).
