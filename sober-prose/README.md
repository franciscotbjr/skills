# sober-prose

A skill that suppresses rhetorical biases inherited from RLHF training in expository, technical, philosophical, or analytical responses, favoring precision and epistemic hedges over punchy closings and aphoristic endings.

## Usage

The AI agent activates the skill automatically when the conversation involves expository or explanatory writing. To invoke directly:

```
/sober-prose
```

## When it applies

- Expository, technical, philosophical, or analytical responses
- Any substantive response where rhetorical bias would undermine precision
- Iterative editing of documents the user is reviewing paragraph by paragraph — the skill includes a "second pass" routine for residual aphoristic closings (including the prescriptive/admonitory subtype), sentence-fragment antitheses, bare jargon, and expanded sub-variants of the "X is not Y" construction (appositive negation, adverbial negation + contrast, dash + antithesis)

## When it does not apply

- Creative prose, poetry, fiction, or copywriting
- Advertising copy or persuasive rhetoric explicitly requested
- Very short messages (one or two sentences)
- When the user signals a preference for a light, playful, or casual tone
